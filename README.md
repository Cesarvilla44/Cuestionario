# 📚 Plataforma de Cuestionarios Interactivos

Sistema completo de cuestionarios educativos con 8 módulos diferentes, diseñado para evaluación de conocimientos técnicos y profesionales.

## 🎯 Cuestionarios Disponibles

1. **Inglés** - Hardware, software, redes, protocolos
2. **Desarrollo Web Servidor** - PHP, bases de datos, protocolos web
3. **Desarrollo Web Cliente** - JavaScript, DOM, AJAX, eventos
4. **Despliegue Web** - Servidores, DNS, FTP, seguridad, LAMP/XAMPP
5. **Empleabilidad II** - Competencias profesionales, metodologías ágiles, liderazgo
6. **Digitalización** - Transformación digital, IA, Big Data, Cloud Computing, IoT
7. **Sostenibilidad** - Economía circular, ODS, RSC, desarrollo sostenible
8. **Ciberseguridad** - Seguridad informática, redes, malware, firewalls

## 📱 Cómo Acceder desde Móvil

### Opción 1: GitHub Pages (Recomendado)
1. Sube todos los archivos a un repositorio GitHub
2. Ve a Settings > Pages
3. Selecciona "Deploy from a branch" > "main"
4. Accede desde: `https://[tu-usuario].github.io/[repositorio]/main.html`

### Opción 2: Servidor Local
1. Instala XAMPP/WAMP o usa Python: `python -m http.server 8000`
2. Desde móvil en la misma WiFi: `http://192.168.1.XXX:8000/main.html`

## 🔄 Cómo Compartir con Compañeros

### Para Compartir el Enlace:
- **GitHub Pages**: Comparte el enlace público
- **Servidor Local**: Comparte tu IP local y puerto

### Para Compartir los Archivos:
1. **GitHub**: Compañeros pueden clonar el repositorio
2. **ZIP**: Comprime la carpeta y compártela
3. **Drive/Cloud**: Sube los archivos a Google Drive, Dropbox, etc.
